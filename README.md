### Domain

- [x] 승리 조건을 판단한다.

### InputView

- [x] 위치를 입력한다.

- [x] 게임 시작 문구를 출력한다.
- [x] 오목 판을 출력한다.
- [x] 돌을 표시한다.

- [X] 진행 중인 게임이 있다면 이전 기록을 불러온다.
- [X] 돌을 둔다.
- [X] 오목인지 판단한다.
- [X] 금수인지 판단한다.
- [X] 오목이라면 더 이상 수를 둘 수 없다.

  - [X] 보드를 초기화한다.
  - [X] 돌을 검정돌로 초기화한다.
