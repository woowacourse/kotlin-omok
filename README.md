# kotlin-omok
- [x] 오목판에 돌을 둘 수 있다.
- [x] 흑이 먼저 돌을 둔다
- [x] 흑과 백이 번갈아 가면서 둔다
- [x] 한 줄로 5개 돌이 연속되어 있을 경우 승리한다 
- [x] 바로 전에 돌을 둔 좌표를 표기한다
- [x] 오목판이 꽉 차면 게임이 종료된다.
- [x] 돌의 좌표를 입력받는다.
- [x] "오목 게임을 시작합니다" 문구 출력.
- [x] 오목판 출력
- [x] 차례와 이전 좌표를 출력
- [x] 33룰
- [x] 44룰
- [x] 6목 체크
- [x] 승자 출력
- [x] 입력받은 X좌표는 A에서 O까지이다.
- [x] 입력받은 Y좌표는 1에서 15까지이다.
